Publication formats

We have a few ways of getting things on the web.

Static sites/pages
Standalone, responsive web apps hosted on the IG server.
IFrames
Rarely used now, but content can be embedded in an iframe inside an FT.com article.
Falcon wrapper pages
Wrappers are FT.com template pages that we can inject our content into.
Heroku apps
For projects that have a significant serverside element.
