ST3 is not free software, but it can be used for free indefinitely (if you don't mind it nagging you occasionally). It's technically still in beta, but it's stable enough.

Installation

Mac OS X

Use Homebrew Cask:

$ brew cask install sublime-text3

Alternatively, go to sublimetext.com/3 and install it manually.

Windows

Go to sublimetext.com/3 and install it manually.

Install some packages

ST3 has ‘packages’, which add extra functionality (such as syntax highlighting for extra languages it doesn't ship with).

  1. Install Package Control.
  2. Now use it to install whatever packages you need.

Recommended packages

These packages are good for our projects:

Babel
Emmet
GitGutter
Handlebars
MarkdownEditing
Sass
SublimeLinter
SublimeLinter-contrib-eslint
SublimeLinter-contrib-scss-lint

To install a package, press CMD+SHIFT+P and start typing "install" (until "Package Control: Install Package" is selected), press ENTER. Wait for the package list to appear, type the one you want, and press ENTER again.
