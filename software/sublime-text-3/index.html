Sublime Text 3

Fast and reliable text editor


sublimetext.com/3

ST3 is not free software, but it can be used for free indefinitely (if you don't mind it nagging you occasionally). It's currently in beta, but it's stable enough.

Installing

Using Cask:

$ brew cask install sublime-text3

Alternatively, go to sublimetext.com/3 and download it.

Get the subl command

TKTKTK

Install some packages

ST3 has 'packages' that add extra functionality (such as syntax highlighting for languages that aren't supported out of the box). You should install Package Control, which in turn will help you install packages.

  1. Install Package Control.
  2. Now use it to install whatever packages you need.
